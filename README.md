## Methods of Artificial Intelligence Directory

1. Framsticks
```
Proszę przeprowadzić eksperyment z wykorzystaniem platformy Framsticks polegający na uruchomieniu procesu ewolucji ukierunkowanej na pewną wybraną cechę (np. szybkość poruszania się, wysokość itp.).

W opracowaniu należy zamieścić opis konfiguracji symulatora, genotypy i wizualizację osobników od których rozpoczynał się proces ewolucji (powinny to być samodzielnie utworzone osobniki, ewentualnie zmodyfikowane wersje osobników z przykładów), genotypy i wizualizację osobników końcowych, wykresy z wynikami (najgorsze, najlepsze oraz średnie przystosowanie populacji) oraz komentarze do wyników.
```
2. Bots
```
Proszę napisać bota, który będzie informował użytkownika o pewnej firmie (produktach/usługach/celach itp.).

Bot powinien mieć 20-30 wzorców w języku polskim i zawierać co najmniej 5 różnych konstrukcji (takich jak redukcja symboliczna, kontekst, losowanie itp.). W komentarzu na górze pliku .aiml proszę wpisać które konstrukcje zostały użyte.
```
